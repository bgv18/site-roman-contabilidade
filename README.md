# Roman Site Contabilidade

Site estático institucional feito com Astro e Tailwind CSS.

## Como executar

1. Instale o Node.js 20+.
2. Rode `npm install`.
3. Rode `npm run dev` para desenvolvimento.
4. Rode `npm run build` para gerar a versão final.

## Estrutura

- `src/pages/index.astro`: landing page principal.
- `src/layouts/BaseLayout.astro`: layout base com meta tags e fontes.
- `src/styles/global.css`: Tailwind e estilos globais.
